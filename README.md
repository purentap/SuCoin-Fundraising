# SUCoin-Fundraising


## To Run Frontend:
### In frontend_react 

  run: npm i (if it does not work run npm i --legacy-peer-deps)

### In /src/contracts_hardhat 

  run:
  
  npm install --save-dev hardhat
  
  npx hardhat compile

### In frontend_react 

  run: npm start

## To Run Backend:
### In backend_dotnet 

  run: dotnet run
